# Gmail QA

Gmail QA is a Chrome extension for email authentication and deliverability analysis inside Gmail.

Features:

* SPF Detection
* DKIM Detection
* DMARC Detection
* Message-ID Inspection
* Broken Message-ID Detection
* Original Message-ID Detection
* List-Unsubscribe Detection
* Unsubscribe Link Extraction
* Tracking Pixel Detection
* Logo Detection
* Quick Copy Actions
* Real-Time Gmail Monitoring

The extension operates entirely within the user's browser and does not transmit email content to external servers.
