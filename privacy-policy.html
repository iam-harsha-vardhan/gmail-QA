Privacy Policy

Last Updated: June 2026

Gmail QA is a Chrome extension designed to help users inspect email authentication and deliverability indicators directly within Gmail.

Information Processing

Gmail QA analyzes information from emails currently opened by the user in Gmail in order to display authentication and deliverability indicators such as:

  • SPF status
  • DKIM status
  • DMARC status
  • Message-ID information
  • Original Message-ID information
  • List-Unsubscribe information
  • Unsubscribe links
  • Tracking pixels
  • Email branding assets and related metadata

Data Collection

Gmail QA does not collect, store, transmit, sell, rent, or share email content, personal information, authentication data, or message metadata with any external server or third party.

Local Processing

All analysis is performed locally within the user's browser. Email data remains on the user's device and is not transmitted outside of Gmail and the user's browser session.

Data Sharing

Gmail QA does not sell user data, transfer user data to third parties, use user data for advertising, or use user data for analytics purposes.

Storage

The extension uses Chrome storage only to save user preferences and settings related to extension functionality.

Third-Party Services

Gmail QA does not load remote code and does not rely on third-party analytics, advertising, or tracking services.

Contact

If you have questions regarding this Privacy Policy, please contact the extension publisher through the contact information provided in the Chrome Web Store listing.
